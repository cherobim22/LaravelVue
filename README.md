## Passos para teste
- No arquivo .env setar as credenciais do seu banco de dados

- Instalando as dependencias do composer
```sh
$ composer install
```
- Realizando as migrations
```sh
$ php run migrate
```
- Instalando as dependencias do node
```sh
$ npm install
```
- Gerando a chave do app
```sh
$ php artisan key:generate
```
